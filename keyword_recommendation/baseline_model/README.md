## baseline01 초기모델

### 📌 1. 개요
블로그 텍스트 데이터를 기반으로<br>
TF-IDF를 활용하여 핵심 키워드를 추출하고,<br>
Word2Vec을 활용하여 연관 키워드를 추천
<br>
<br>

### 📁 2. 데이터 파일 설명
- `baseline01.ipynb`
  - 초기 모델 파이썬 코드 파일
- `result_text.csv`
  - 블로그 게시물 수동 크롤링 데이터
<br>

### 💻 3. 과정
**1. 텍스트 전처리**
- `KoNLPy Okt`를 사용하여 명사 추출
- 텍스트 데이터를 키워드 분석에 적합한 형태로 변환

**2. 핵심 키워드 추출**
- `TF-IDF`를 활용하여 상위 30개의 중요 키워드 선정

**3. 연관 키워드 추천**
- `Word2Vec` 모델 학습
- 각 핵심 키워드별 유사 단어 Top N개 추출
<br>

### 📊 4. 결과
- TF-IDF 기반 핵심 키워드 리스트 생성
- Word2Vec 기반 연관 키워드 추천 결과 도출
<br>

### ❗️ 5. 한계
- 명사 중심 분석으로 문장 단위 의미 반영 어려움
- TF-IDF 기준으로 단순 빈도 기반 중요도 계산
- 사용자 반응이 키워드 추출에 반영되지 않음
<br>

### 💡 6. 개선 방향
- **사용자 반응 데이터 반영**
- 불용어 제거 및 필터링 강화 등 전처리 고도화
- LDA 토픽 모델링을 활용하여 키워드 -> **주제 단위 분석**으로 확장
