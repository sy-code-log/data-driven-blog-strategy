## baseline01 초기모델

### 📌 1. 개요
초기 모델에서는 블로그 텍스트 데이터를 기반으로 핵심 키워드를 추출하고,<br>
Word2Vec을 활용하여 연관 키워드를 추천하는 방식을 적용
<br>
<br>

### 💻 2. 과정
**1. 텍스트 전처리**
- `KoNLPy Okt`를 사용하여 명사 추출

**2. 핵심 키워드 추출**
- `TF-IDF`를 활용하여 상위 30개의 중요 키워드 선정

**3. 연관 키워드 추천**
- `Word2Vec` 모델 학습
- 각 핵심 키워드별 유사 단어 Top N개 추출
<br>

### 📊 3. 결과
- TF-IDF 기반 핵심 키워드 리스트 생성
- Word2Vec 기반 연관 키워드 추천 결과 도출
<br>

### ❗️ 4. 한계
- 명사만 사용하여 문맥 정보 손실
- TF-IDF 기준으로 단순 빈도 기반 중요도 계산
- 사용자 반응이 키워드 추출에 반영되지 않음
<br>

### 💡 5. 개선 방향
- **사용자 반응 데이터 반영**
- 전처리 과정 개선 (불용어 제거, 필터링 강화)
