## 시간대별 조회수 분석

### 📌 1. 프로젝트 개요
사용자 반응 데이터를 기반으로 블로그 게시글의 **시간대별 조회수 패턴**을 분석하고, <br>
이를 통해 **최적의 콘텐츠 업로드 시간 전략을 도출**하는 것이 목표
<br>
<br>

### 📁 2. 데이터 파일 설명
- `각 시간대별 파일.zip`
  - 원본 데이터
  - 각 시간대별 csv 파일 안, 날짜별 조회수 합계 데이터
- `time_all.csv`
  - 원본 데이터를 모두 concat한 데이터
- `시간대별조회수.twb`
  - Tableau 기반 시각화 대시보드 파일
  - 시간대별 조회수 패턴 확인
<br>

### 💻 3. 분석 방법
- 시간대 기준으로 그룹화
- 시간대별 조회수 합계 계산
- 선 그래프로 결과 출력 (Tableau)
<br>

### 📊 4. 주요 결과
- 특정 시간대에 조회수가 집중 (14 ~ 16시) 
<br>

### 💡 5. 인사이트
- 블로그 지수는 게시 직후 초기 반응에 영향
- 조회수 높은 시간대에 콘텐츠 업로드 추천
